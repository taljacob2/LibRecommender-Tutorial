## Setup

1. Create virtual environment:

   ```
   python3 -m venv .venv
   ```

1. Activate venv

   - In Windows:

     ```
     . .venv/Scripts/activate
     ```

   - In Linux:

     ```
     . .venv/bin/activate
     ```

1. Install packages from `requirements.txt`:

   ```
   pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt
   ```
